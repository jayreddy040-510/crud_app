API Endpoints

Below is a list of available API endpoints and their functions:

  • GET /products/ - List all products.
  • GET /invoices/ - List all invoices.
  • GET /invoice-items/ - List all invoice items.
  • GET /customers/{pk}/ - Retrieve details of a customer (replace {pk} with actual customer ID).
  • POST /invoices/new/ - Create a new invoice.
  • POST /invoice-items/{pk}/update/ - Update an existing invoice item (replace {pk} with actual invoice item ID).
  • POST /products/{pk}/delete/ - Delete a product (replace {pk} with actual product ID).
  • GET /invoice-items/invoice/{invoice_id}/ - List all invoice items for a specific invoice (replace {invoice_id} with actual invoice ID).
  • GET /invoice-items/invoice-no/{invoice_no}/ - List all items of an invoice by invoice number (replace {invoice_no} with actual invoice number).
